# React Notes App

This React Notes App is built using the following technology stack:

1. **React 19**
2. **React Context**
3. **Reusable components**
4. **Saving and reading notes in Localstorage**

### Installation and Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the application:

```bash
npm run dev
```
